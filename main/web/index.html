Контроллер держит свою точку BrewControl и одновременно подключается к домашнему роутеру. После подключения к роутеру он доступен по IP в общей сети.

Контроллер сам качает новую прошивку из GitHub Releases по HTTPS, проверяет образ и обновляется. При сбое — автоматический откат.
Обновление файлом (локально)
загрузка...
Выбери файл build/brewcontrol.bin. После загрузки контроллер проверит образ, перезапишет неактивный слот и перезагрузится. Если новая прошивка не стартует — автоматически откатится на предыдущую.
